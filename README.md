# ClassWatch

Sistema que grava aulas em áudio, transcreve automaticamente usando IA e gera resumos organizados para os alunos

## Funcionalidades

- Gravação de aulas em áudio
- Transcrição automática usando IA
- Geração de resumo em tópicos
- Armazenamento das aulas e conteúdos

## Tecnologias

- Python
- Whisper (IA para transcrição)
- FFmpeg (processamento de áudio)
- Java (backend - futuro)
- Banco de dados (em desenvolvimento)

## Como rodar

1. Instale o Python
2. Instale as dependências: pip install openai-whisper
3. Instale o FFmpeg
4. Execute o script: python transcricao.py

## Estrutura

- /Código → scripts principais
- /Audios → arquivos de entrada
- /Teste_transcricao → saída de transcrições

## Uso

1. Coloque o áudio na pasta
2. Execute o script
3. A transcrição será gerada automaticamente
